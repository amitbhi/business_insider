# 🧠 BUSINESS_INSIDER

## Sovereign, Multi-Agent AI Intelligence Platform (India V1)

An intelligence engine that converts raw public and corporate regulatory data into actionable strategic intelligence via highly specialized, independent AI agents governed by a centralized local orchestration mechanism.

### Core Modules
*   **Finance Agent:** Corporate balance sheets, debt loads, structural vulnerabilities.
*   **Power / Control Agent:** Directors, Ownership maps, UBO detection.
*   **Legal Agent:** Risk, lawsuits, statutory penalties.
*   **Market Agent:** Synthesis, dependencies.
*   **Social Agent:** Probabilistic relationship clustering, familial mapping.

### Technology Stack
*   **Orchestration:** Python (Custom Multi-Agent Framework) + FastAPI
*   **Local LLM:** Ollama (DeepSeek)
*   **Knowledge Persistence:** PostgreSQL + NetworkX Edge Logic
*   **Visualization:** Streamlit + PyVis

### System Execution Lifecycle
1.  **Ingestion:** API receives target (e.g. "Reliance Industries").
2.  **Orchestration:** Agent Manager triggers the worker modules.
3.  **Graph Synthesis:** Entity Models, Nodes, and Relationships are compiled deeply.
4.  **Strategic Reasoning:** Strategic Engines surface Weak Founder alerts or Power concentration clusters. 
5.  **Retrieval:** UI displays interactive Network Graph alongside human-readable intelligence warnings.
